🎨 Code Style

TypeScript

  • NO any types — use unknown
  • NO as casts — use type guards or Zod
  • Explicit return types on exported functions

Naming

  • Components: PascalCase → ChatWindow.tsx
  • Utils/hooks: camelCase → useSession.ts
  • Constants: UPPER_SNAKE_CASE

Imports

import { getUser } from "@/lib/data-layer";  // ✓
import "../utils";                               // ✗

🔐 Secret Governance (Two-Step)

Step 1: Entry Validation

  • Runs at app startup via instrumentation.ts
  • Validates all secrets exist and meet minimum length
  • Cross-checks: JWT_SECRET ≠ PROVENANCE_SIGNING_KEY
  • Fails fast on critical errors

Step 2: Cleanup

  • Called on SIGTERM/SIGINT/uncaughtException
  • Scrubs secrets from memory
  • Prevents leak in crash dumps

🛡️ Safety Rules

  • NEVER Weaken rate limiting or auth
  • NEVER Log user text content
  • NEVER Expose secrets in responses
  • ALWAYS Fail-closed on safety exceptions
  • ALWAYS Validate input with Zod
